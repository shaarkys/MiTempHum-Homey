# MiTemp_BLE

Aplicación para el Reloj de Monitoreo de Temperatura y Humedad de Xiaomi LYWSD02MM, Xiaomi Mijia LYWSD03MMC y el sensor de temperatura y humedad Bluetooth Xiaomi MiJia.
