# MiTemp_BLE

Aplikacja dla urządzeń Xiaomi Temperature and Humidity Monitor Clock LYWSD02MM, Xiaomi Mijia LYWSD03MMC Temperature and Humidity oraz czujnika temperatury i wilgotności Xiaomi MiJia Bluetooth.
