Следите за окружающей средой: Подключите датчики температуры и влажности Mijia BLE (LYWSD03MMC и LYWSDCGQ/01ZM) без труда.

Обратите внимание, что датчик LYWSD03MMC необходимо перепрошить пользовательской прошивкой 'ATC' с использованием рекламного формата ATC441.
Для этого посетите https://github.com/pvvx/ATC_MiThermometer и используйте Telink Flasher для прошивки текущей поддерживаемой версии v3.5.
После прошивки не забудьте установить рекламный формат ATC1441.
Приложение отфильтрует обнаруженные устройства по формату их данных (0x181A) и выведет список устройств по имени.
